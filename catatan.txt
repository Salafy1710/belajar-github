GIT RESET
Perumpamaan : 
working directory : kertas kosong
staging area : kertas berada di meja guru
commit : kertas di simpen di meja loker guru 

mode :
- soft : kertas di kembalikan ke meja guru artinya perubahan di kembalikan ke staging index
- mixed : kertas di kembalikan ke anak tersebut artinya perubahan kembali ke working directory
- hard : kertas di robek guru artinya perubahan benar2 dihapus / direset
